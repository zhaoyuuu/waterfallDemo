    实现窗口大小变化Container组件重新渲染，但是Container组件之前的数据（Ajax请求的次数）怎么保存？？
    >解决方法：在父组件中初始化count，Container子组件每请求一次就跟父组件通信要求count++。窗口大小改变，子组件销毁再重新挂载，挂载时父组件通知子组件count的值，子组件立马请求count次ajax请求。
